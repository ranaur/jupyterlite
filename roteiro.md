# Commands for local execution

## Setup environment


```
pip -m venv env
source env/bin/activate
```

Replace EXEs as python scripts

```
./python-stub
```

## Rebuild exe as python scripts

Builds in _output directory

```
jupyter-lite build
```

Serve site:

```
jupiter-lite serve
```
